delta = rt + 0.95 * V(t+1) - V(t)

d3: "Accidentally Path own creature"
delta = rt + 0.95 * V(t+1) - V(t)
delta = 0 + 0.95 * (+0.1) - (+0.5)
delta = -0.405
BAD: Made things worse, even in a winning game
GAE propagates advantages right-to-left with lambda=0.90 decay, so good/bad decisions get credit even many turns before the outcome. Formula: A_t = sum of (gamma*lambda)^l * delta_{t+l}
