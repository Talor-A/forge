4. Training Pipeline

1
Data Collection — 1,000 heuristic AI vs AI games, 16 threads, capturing 28,828 decisions via EventBus
forge rltrain collect -d deck1.dck -d deck2.dck -n 1000 -t 16
2
Value Network Training — Predict win/loss from any mid-game state. 100 epochs, 99.6% accuracy
python training/training_ui.py --epochs 100 --device cuda
3
Decision Head Training — Freeze encoder, train attack (13,918 decisions) + block (6,546 decisions) heads
python training/train_decisions_ui.py --epochs 50 --device cuda
4
Model Server + Evaluation — Serve trained model, run RL agent vs heuristic AI, measure win rate
python serving/model_server.py --device cuda
5
PPO Self-Play — RL self-improvement beyond heuristic level with curriculum card pools
Population-based training with Elo tracking
6
Progressive Scaling — Grow model 11M → 50M → 150M with weight transfer, expand card pool
ONNX export for Java-native deployment
